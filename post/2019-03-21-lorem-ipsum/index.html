Short article about setting up my personal webpage for blogging.

25 March 2019
test computer web

Setup

I wanted to setup my webpage for posting articles and thoughts. I was wandering over the vast plains of internets when I stumbled upon GitHub Pages. Since I have a GitHub account I thought to myself why wouldn’t I make myself a webpage. But how? I know some programming – I do some scripting in bash and python and I do know some html, css and JavaScript, which should be sufficient for a web page. But I didn’t know where exactly to start, so I googled how to setup my first webpage and so I came across Hugo.

Hugo

Hugo is a fast and modern static site generator written in Go, and designed to make website creation fun again. So, that is what they say on their page.

Because I really don’t have much (or any for that matter) experience in building websites, I thought to myself: well, why not Hugo then. There are more than 100 different themes you can use and some of them are really nice. For my site I wanted minimalistic feel, so I decided to start with the cactus theme. I did some tweaks here and there to get the feeling right and now I am writing my first article, which brings me to content.

Content

What will I publish here? Should I have a content theme for my site? Well, I think I’ll start with the most obvious thing: my work and my interests. I guess I’ll mostly write about what I do for work and my PhD, however, I will not limit myself. If there will be something I’ll think is worth writing about, then I’ll still do it.

Conclusion

So, here I am.
I managed to get myself a webpage with custom domain name. I will probably rewrite this article once I’ll have more time, but until then, this will have to do it.
